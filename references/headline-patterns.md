# Headline Patterns

Use this reference before choosing final in-image text.

## Priority Ladder

1. Practical investor question.
2. Market consequence.
3. Asset class or topic.
4. Specific ticker/company only when central.
5. Technical detail only when the audience expects it.

## Audience Modes

### Mass Investor

Good:

- `ОСНОВЫ БИРЖЕВОЙ ТОРГОВЛИ`
- `КАК СЧИТАТЬ РИСКИ`
- `АКЦИИ ИЛИ ОБЛИГАЦИИ`
- `ПОРТФЕЛЬ БЕЗ ЛИШНЕГО РИСКА`

Avoid:

- terminal jargon;
- fake urgency;
- promises of profit.

### Market-Aware

Good:

- `ТОРГОВАЯ ИДЕЯ НЕДЕЛИ`
- `ЧТО ПРОИСХОДИТ С РЫНКОМ`
- `ГДЕ РИСК, А ГДЕ ВОЗМОЖНОСТЬ`
- `ОБЛИГАЦИИ ПОСЛЕ СТАВКИ`

### Professional / Technical

Allowed when relevant:

- `MOEX`
- ticker names;
- `P/E`, `EV/EBITDA`;
- yield, duration, risk premium;
- technical analysis terms.

Still keep image text short and readable.

## Formulas

- `<ТЕМА> БЕЗ <БОЛЬ>`
- `КАК <ДЕЙСТВИЕ> <РЕЗУЛЬТАТ>`
- `<АКТИВ> ИЛИ <АКТИВ>`
- `<РЫНОК> ПОСЛЕ <СОБЫТИЕ>`
- `<ОШИБКА> КОТОРАЯ СТОИТ ДЕНЕГ`

## Avoid

- `100%`
- `гарантия`
- `иксы`
- `ракета`
- `срочно покупай`
- `разбогатей`
- fake numbers;
- tiny source lines.
