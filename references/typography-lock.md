# Birzhevik Typography Lock

Use this reference before every Birzhevik cover.

## Current Font Direction

The supplied identity PDF names:

- `Vela Sans GX`
- `Manrope`

Target traits:

- clean modern Cyrillic grotesk;
- white display typography on dark navy/black surfaces;
- confident large headline, often left-weighted;
- readable line length and strong safe-zone margins;
- blue support line or secondary text when useful.

## Rules

- Use Vela Sans GX for headline direction when the asset is available.
- Use Manrope for support text, chips, captions, and secondary labels.
- If exact font files are unavailable, ask GPT Image 2 for Vela Sans GX / Manrope-like typography and keep Russian text short.
- If exact text fidelity matters, generate the scene with clean text space and add typography manually using official font files.
- Keep all-caps headlines tracked enough to breathe.
- Do not stretch letters vertically.
- Do not use ultra-condensed, techno, serif, handwritten, or meme fonts.

## Quality Gate

Reject or correct when:

- the headline looks like a generic YouTube trading thumbnail;
- Russian text is misspelled or warped;
- the type is too thin for a thumbnail;
- the caption is trying to carry the main idea;
- the headline sits too close to an edge or over a busy chart.
