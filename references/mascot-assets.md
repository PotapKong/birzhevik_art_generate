# Mascot / Character Rules

Birzhevik currently has no mandatory mascot in the supplied identity PDF.

## Current Rule

Do not invent a permanent character, robot, trader avatar, bull, bear, or mascot unless the user supplies one or explicitly asks for a character direction.

Use brand objects instead:

- Birzhevik logo token;
- glass logo tile;
- metallic coin stack;
- microphone;
- market chart arrow;
- abstract financial surface.

## Future Mascot Intake

If a future mascot is supplied:

1. Add the official asset under `references/images/` or `assets/brand/`.
2. Update this file with identity traits.
3. Require the mascot to be integrated into the generated scene, not pasted as a sticker.

Until then, the brand mark/token is the main identity carrier.
